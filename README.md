# Windows-Activator

A simple, lightweight Windows 10/11 activator with a clean dark GUI. Supports all editions (Home, Pro, Enterprise, Education, and their N variants).

## Features

- ✅ Activates Windows 10/11 (all editions)
- ✅ Dark cyber GUI
- ✅ One-click activation
- ✅ No external tools required
- ✅ Checks if Windows is already activated
- ✅ 100% free

## How to Use

1. **Download** the latest release or run the Python script.
2. **Run as Administrator**.
3. **Select your Windows version** from the dropdown.
4. Click **"ACTIVATE WINDOWS"**.
5. Wait for the process to complete.

---

## Requirements

- Windows 10 or 11
- Python 3.9+ (if running from source)
- Internet connection (for KMS activation)
